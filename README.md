# AgentPractice

This repository contains practical exercises and projects for learning AI Agents.

## Structure
- **Phase 1: Foundations** - Basic LLM API integration and prompt engineering exercises.
- **Phase 2: Core Concepts** - Implementations of ReAct, RAG, and Memory systems.
- **Phase 3: Frameworks** - Projects using LangChain, AutoGen, and CrewAI.
- **Phase 4: Advanced** - Multi-agent systems and complex deployments.

## Getting Started
1. Clone the repository.
2. Install dependencies (see individual project folders).
3. Set up your `.env` with necessary API keys.
